## 🎭 ANIMACIONES CON FRAMER MOTION

### Hover Effects
- **Botones:** scale(1.05) + cambio de color suave, transición 0.3s ease
- **Tarjetas (blog, empresas):** sombra suave + scale(1.02), transición 0.3s
- **Links:** cambio de color suave, transición 0.2s
- **Foto hero:** scale(1.03) + glow sutil verde oliva, transición 0.4s

### Scroll Animations (Framer Motion)
- **Fade in + translateY:** Cada sección entra con fade y sube 20px
  * Duración: 0.6s, ease-out
  * Delay escalonado entre elementos: 0.1s
  * Trigger: cuando sección entra a viewport (useInView)

### Microinteracciones
- **Formulario contacto:** spinner suave al enviar
- **Botones CTA:** pulso sutil al hover (pulse opacity 0.5)
- **Scroll smooth:** transiciones suaves entre secciones
- **Números en secciones:** contador animado (0 → número final)

### Efectos especiales (opcionales)
- **Parallax suave:** secciones de fondo se mueven ligeramente con scroll (factor 0.5)
- **Líneas verdes:** animación de entrada (width: 0 → 100%)
- **Fade de fondos:** transición suave de color al cambiar de sección en scroll
